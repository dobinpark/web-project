# 매일데이(Maeil Day)

직장인들의 새로운 활력소,
이곳이 바로 매일데이 커뮤니티입니다.

# 💡 Topic
* 직장인들을 대상으로 지루한 일상속에 서로 소통하며 공감대를 형성할 수 있는 소통의 장으로 만든 서비스
* 자신만의 굿즈를 제작하여, 매일데이에서 직접 상품을 등록, 대행판매를 할 수 있는 서비스
* 직장인들의 점심을 어디에서 먹을지 고민할 때 검색하면 주변의 맛집 리스트를 보여주는 서비스

# 📝 Summary
대학생들의 커뮤니티 서비스인 에브리타임을 보면서 ‘직장인들을 위한 에브리타임은 없을까?’라는 생각을 했습니다.

에브리타임처럼 깔끔하고 직관적인 커뮤니티를 직장인들에게 제공한다면, 보다 편하게 서로 소통하면서<br>
매일데이라는 커뮤니티 서비스를 일상 속에서 언제든지 이용할 수 있도록 만든 서비스입니다.

# ⭐️ Key Function
* 매일게시판
  * 공지게시판, 자유게시판, 질문게시판등 다양한 게시판을 통해 게시판의 기능을 사용할 수 있음.
  
* 매일몰
  * 직장인들의 창의성을 발휘하여 매일몰에 상품을 등록하고 매일데이에서 대행 판매할 수 있음
  
* 매일 맛집
  * 회사 근처에 맛집 리스트를 제공하여 점심 고르는 시간을 단축할 수 있는 편의성을 제공할 수 있음

# 🤔 Learned
* 첫 웹 애플리케이션 프로젝트를 개발하면서, 생각보다 많은 비즈니스 로직과 라이브러리가 필요하다는 것을 알게 되었음.
* 많은 데이터를 다루다 보니 DB 설계의 중요성을 깨닫게 됨.
* 추후에 서비스를 확장할 예정인데 단순히 기능 구현에 초점이 아닌, 유지보수를 위해서 리팩토링과정을 통해 불필요한 코드를 개선하고<br>
   보다 효율적인 클린 코드로 웹 서비스를 개발해야겠다는 생각이 듦.
* 이번 프로젝트에서는 Spring Data JPA을 활용하여 실무에 적용하는 기술을 학습할 수 있었음..
* AWS 배포를 목적으로 AWS 학습를 하고, 배포를 위해 보안 부분에 학습해볼 수 있었음.
* 혼자 개발하다 보니 프론트엔드, 백엔드의 역할 분배의 필요성을 느끼게 됨.
